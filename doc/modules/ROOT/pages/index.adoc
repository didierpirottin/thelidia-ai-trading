= Base de Connaissance — Investissement en Bourse
:description: Vue d'ensemble de la base de connaissance : actions, options, stratégies, hedging, portefeuille

Bienvenue dans cette base de connaissance dédiée à l'investissement en bourse avec les options.
Elle couvre l'ensemble des concepts — des fondamentaux aux stratégies avancées — organisés par thème.

TIP: Vous débutez ? Commencez par xref:options/index.adoc[Options] pour comprendre les bases, puis xref:strategies/global-strategy.adoc[la Stratégie Globale] pour une vue d'ensemble.

== Par où commencer ?

----
  VOTRE PROFIL                              POINT D'ENTRÉE RECOMMANDÉ
  ─────────────────────────────────────────────────────────────────────

  Je débute complètement
  └── Comprendre ce qu'est une option       → Options — Concepts fondamentaux

  Je connais les options, je veux une méthode
  └── Approche structurée par phase de marché → Stratégie Globale

  Je veux générer des revenus mensuels
  ├── Avec des actions que je possède       → Wheel / CSP
  └── Sans détenir d'actions               → Rolling Put Diagonal

  Je veux protéger mon portefeuille
  ├── Une seule ligne                       → Protective Put / Collar
  └── Portefeuille diversifié              → Couverture par indices (SPY / QQQ)

  J'ai une position en perte
  └── Abaisser le breakeven sans capital    → Stock Repair Strategy
----
